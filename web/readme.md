## Bootstrp4网格系统
### 网格类
需要记住：网格系统要求每个行最多12个列,并且需要'''.row'''类进行标记<br>
#### 网格系统有5类
'''.col-x'''>>>>>>>针对所有设备<br>
'''.col-sm-x'''>>>>针对平板设备<br>
'''.col-md-x'''>>>>针对桌面显示器<br>
'''.col-lg-x'''>>>>针对大桌面显示器<br>
'''.col-xl-x'''>>>>针对超大桌面显示器v
如果没有表明网格系统类的话，那么Bootstrap会自动排列<br>
